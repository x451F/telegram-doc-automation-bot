# Templates Directory

Store generic `.docx` templates in this folder.

Current demo templates:

- `contract_template.docx`
- `completion_certificate_template.docx`

Supported placeholder style:

- `[contract_number]`
- `[contract_date]`
- `[city]`
- `[contract_total_amount]`
- `[net_amount]`
- `[certificate_number]`
- `[certificate_date]`
- `[certificate_amount]`
- `[certificate_amount_text]`
- `[contract_work_1]` ... `[contract_work_5]`
- `[certificate_work_1]` ... `[certificate_work_5]`
